## Installation
Buy the game from GOG or Steam and place the data.solarus file in the oceansheart/game folder.

## Default Gameplay Controls
| Button | Action |
|--|--|
|D-PAD|Move|
|A|Action / Dodge|
|B|Attack|
|X|Item 1|
|Y|Item 2|
|L1|Previous Menu|
|R1|Next Menu|
|START|Menus|
