## Installation
Download the game from https://solarus-games.org/games/the-legend-of-zelda-book-of-mudora/ and put in the zelda-bookofmudora/game folder.

## Default Gameplay Controls
| Button | Action |
|--|--|
|D-PAD|Move|
|A|Interact|
|B|Attack|
|X|Item 1|
|Y|Item 2|
|START|Menus|
